# 🚀 URL-Shortner — Premium URL Management

A modern, high-performance URL shortening platform with a **premium dark-mode aesthetic**, real-time analytics, and glassmorphism design.

![Banner](final_mobile_ui_1774804632193.png)

## ✨ Features

-   **⚡ Instant Shortening**: Transform long, cluttered URLs into clean, memorable links.
-   **📊 Real-time Analytics**: Track device types, locations, and click-through rates.
-   **💎 Premium UI/UX**:
    -   Deep-dark theme with vibrant radial gradients.
    -   Glassmorphism UI components for a modern look.
    -   Hidden scrollbars for a clean, immersive experience.
-   **🔐 Robust Authentication**: Secure user login/signup via Supabase.
-   **📱 Mobile First**: Fully responsive design optimized for speed and aesthetics on any screen.

## 🛠️ Tech Stack

-   **Frontend**: React + Vite + Tailwind CSS
-   **Backend**: Supabase (Auth + Database)
-   **Components**: Lucide Icons + Radix UI + Shadcn/UI
-   **Deployment**: Vercel

## 🚀 Getting Started

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/sachin98765/url-shortner.git
    cd url-shortener
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Set Up Environment**:
    Create a `.env` file with your Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_KEY=your_supabase_key
    ```

4.  **Run Dev Server**:
    ```bash
    npm run dev
    ```
